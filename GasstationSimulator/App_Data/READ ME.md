Programm Start:
- Es stehen drei Tabs zur Verfügung: Home, Customer und Supervisor

Tab Customer:
- Es stehen vier Tanksäulen zur Verfügung, standardmässig ist die erste selektiert.
- Jede Tanksäule hat drei Zapfhähne mit je unterschiedlichen Benzinsorten.

		Select (Tap 1/2/3):
		- Wird ein Zapfhahn ausgewählt, so kann man nun eingeben, wie viele Liter man tanken möchte.
		- Mit Refuel wird die Benzinmenge eingetankt und der Preis berechnet.
		- Es kann beliebig oft nachgetankt werden.

		To Checkout:
		- Ist man fertig mit tanken, wird man zur Kassenauswahl weitergeleitet.

		Cash Register (1/2):
		- Ist eine Kasse ausgewählt worden, kann nun gewählt werden, mit wie viel gezahlt wird.
		- Es könne mehrere Noten/Münzen eingegeben werden.

		Pay:
		- Ist genug Geld einbezahlt worden, wird der Rückgabebetrag berehnet und anschliessend zurückgegeben.
		- Es wird eine Quittung erstellt, wo verschiedene Daten nochmal eingesehen werden können.

Tab Supervisor:
- Hier werden interessante Statistikwerte angezeigt.

Tab Home:
- Zurück auf die Startseite
